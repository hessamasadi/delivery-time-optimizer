# Delivery Time Optimizer

Predict delivery time and select optimal vehicle type for food delivery orders.

## Model
- Linear Regression (MAE: 5.92 min, R²: 0.829)
- Trained on 1000 orders with weather, traffic, distance, preparation time, and courier experience

## Features
- Predict delivery time for any order
- Optimize vehicle selection (Bike/Car/Scooter) for given conditions
- Handles missing values with 'Unknown' categories
